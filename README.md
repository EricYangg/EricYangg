## Featured projects

### Heart Failure Classification

End-to-end ML pipeline predicting heart failure risk from clinical data. 86% accuracy, F1 0.88 on the positive class. Reproducible setup with Docker, Makefile, pytest, and GitHub Actions CI/CD. 4-person team, 226 commits across 59 branches.

Repo: [github.com/EricYangg/Heart-Failure-Classification](https://github.com/EricYangg/Heart-Failure-Classification)

### Brain-Robot Interface (peer-reviewed publication)

EEG-based real-time classification of directional thoughts to control a robot. Benchmarked Deep Learning CNNs (TensorFlow), XGBoost, and several Scikit-learn classifiers. 45% accuracy across 5 classes on real-time EEG, well above the 20% random baseline given the noise floor of consumer-grade hardware.

Published in *Sensors & Transducers*, Vol. 257, Issue 3, May 2022, pp. 30 to 37. [Read the paper](https://sensorsportal.com/HTML/DIGEST/P_3266.htm)

### IBM Applied Data Science Capstone

SpaceX Falcon 9 landing success prediction. Full pipeline including SpaceX API and web scraping, feature engineering, classification models (Logistic Regression, SVM, KNN, Decision Tree), and Plotly Dash plus Folium dashboards for results visualization.
