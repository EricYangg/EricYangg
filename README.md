## Featured projects

### Heart Failure Classification

End-to-end ML pipeline predicting heart failure risk from clinical data. **86% accuracy**, F1 0.88 on the positive class. Reproducible setup with Docker, Makefile, pytest, and GitHub Actions CI/CD. 4-person team, 226 commits across 59 branches.

[Repo](https://github.com/EricYangg/Heart-Failure-Classification)

### Brain-Robot Interface (peer-reviewed publication)

EEG-based real-time classification of directional thoughts to control a robot. Benchmarked Deep Learning CNNs (TensorFlow), XGBoost, and several Scikit-learn classifiers. **45% accuracy across 5 classes on real-time EEG**, well above the 20% random baseline given the noise floor of consumer-grade hardware.

Published in *Sensors & Transducers*, Vol. 257, Issue 3, May 2022, pp. 30 to 37.

[Read the paper](https://sensorsportal.com/HTML/DIGEST/P_3266.htm) · [Collab repo](https://github.com/ayyymiel/thebrainboys)

### USA Crime Tracker (MDS Data Visualization)

Interactive R Shiny dashboard exploring historical crime trends across U.S. cities and states. Built as a 4-person team project for UBC MDS DSCI 532.

[Repo](https://github.com/UBC-MDS/DSCI-532_2026_13_usa-crime-tracker)

### IBM Applied Data Science Capstone

SpaceX Falcon 9 landing success prediction. Full pipeline including the SpaceX API and web scraping, feature engineering, classification models (Logistic Regression, SVM, KNN, Decision Tree), and Plotly Dash plus Folium dashboards for results visualization.

[Repo](https://github.com/EricYangg/IBM-Applied-Data-Science-Capstone)
